# Anime & Genshin Impact Web Application

## Project Overview
- **Date**: April 29, 2026
- **Project Type**: Blazor Web Application (.NET 9)
- **APIs Used**: 
  - Jikan API (https://api.jikan.moe/v4/) - Unofficial MyAnimeList API
  - Genshin API (https://genshin.jmp.blue/) - Fan-made Genshin Impact API
- **Database**: SQLite (anime.db)

## Database Structure

### AnimeItems Table
```sql
CREATE TABLE "AnimeItems" (
    "Id" INTEGER NOT NULL CONSTRAINT "PK_AnimeItems" PRIMARY KEY AUTOINCREMENT,
    "MalId" INTEGER NOT NULL,
    "Title" TEXT NULL,
    "ImageUrl" TEXT NULL,
    "Synopsis" TEXT NULL,
    "Episodes" INTEGER NULL,
    "Status" TEXT NULL,
    "WatchStatus" TEXT NOT NULL,
    "Details" TEXT NULL,
    "Rating" INTEGER NOT NULL,
    "ShowDetails" INTEGER NOT NULL
);
```
